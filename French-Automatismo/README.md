# French-Automatismo: Curso Práctico A2 → C1

## 🎯 Descripción del Curso

**French-Automatismo** es un curso intensivo y práctico diseñado para llevarte desde un nivel A2 básico hasta un nivel C1 avanzado en francés oral, enfocándose en el **automatismo del habla** mediante ejercicios hablados, repetición activa, shadowing, drills y práctica cronometrada.

### Características Principales

✅ **Enfoque 100% Práctico**: Mínima teoría, máxima práctica hablada  
✅ **Automatización**: Ejercicios diseñados para eliminar la traducción mental  
✅ **Cronometrado**: Simulación de presión real de conversación  
✅ **Shadowing Intensivo**: Imitación de nativos para mejorar pronunciación y ritmo  
✅ **Anki Integrado**: Repetición espaciada de frases completas (no palabras sueltas)  
✅ **Grabaciones**: Práctica con feedback mediante grabación de voz  
✅ **Ejercicios Interactivos HTML**: Herramientas cliente local para práctica cronometrada

---

## 🎙️ Ejercicios Interactivos (Cliente Local)

Este curso incluye herramientas HTML/JavaScript que funcionan **100% en tu navegador** (sin servidor):

### ejercicio-respuestas-rapidas.html

Herramienta interactiva para practicar respuestas automáticas en francés con:

- ⏱️ **Cronómetro por pregunta**: Control de tiempo de respuesta
- 🎤 **Grabación de audio**: Usa MediaRecorder API para grabar tus respuestas
- ✅ **Evaluación manual**: Marca tus respuestas como Correcto/Incorrecto
- 💾 **Guardado automático**: Los resultados se guardan en localStorage
- 📊 **Exportación CSV**: Descarga tus estadísticas para análisis
- 🗣️ **Reconocimiento de voz** (opcional): Si tu navegador lo soporta (Web Speech API)

#### Cómo usar el ejercicio interactivo:

1. **Abre el archivo HTML**: 
   - Navega a `French-Automatismo/ejercicio-respuestas-rapidas.html`
   - Haz doble clic o abre con tu navegador preferido

2. **Navegador recomendado**:
   - **Chrome** o **Edge** (mejor compatibilidad con MediaRecorder)
   - Firefox también funciona
   - Safari tiene limitaciones con algunas funciones

3. **Configuración inicial**:
   - El navegador pedirá permiso para usar el micrófono
   - Acepta el permiso para poder grabar

4. **Durante el ejercicio**:
   - Lee la pregunta en francés
   - Presiona el botón de grabación
   - Responde en voz alta
   - Detén la grabación
   - Escucha tu respuesta
   - Marca como Correcto/Incorrecto
   - Pasa a la siguiente pregunta

5. **Guardado y descarga**:
   - Tus resultados se guardan automáticamente en localStorage
   - Puedes descargar un CSV con tus estadísticas
   - Los audios quedan guardados localmente (no se suben a ningún servidor)

#### Notas importantes:

- 🔒 **Privacidad**: Todo funciona localmente, ningún dato sale de tu computadora
- 📱 **Sin conexión**: Una vez cargado, funciona sin internet
- 🎯 **Evaluación básica**: La evaluación es manual (tú marcas correcto/incorrecto)
- 🚀 **Extensible**: Se puede ampliar con Web Speech API para reconocimiento automático

## 🎙️ Metodología de Práctica

### Principios Fundamentales

1. **Todo se hace HABLANDO**: Nunca leas solo mentalmente
2. **Graba TODO**: Para poder escuchar y mejorar
3. **Usa cronómetro**: Simula presión real
4. **Repite hasta automatizar**: La fluidez viene de la repetición
5. **Practica 20-30 min diarios**: Mejor que 2 horas una vez por semana

### Técnicas Principales

- **Shadowing**: Imitación simultánea de audio nativo
- **Drills**: Repetición de estructuras hasta automatizarlas
- **Monólogos cronometrados**: Hablar sin parar sobre temas diversos
- **Role-plays**: Simulación de situaciones reales
- **Anki con frases**: Repetición espaciada de chunks completos

## 💡 Consejos para el Éxito

### ✅ Haz SIEMPRE
1. **Practica todos los días** (aunque sean 15 minutos)
2. **Graba tu voz** para poder comparar tu evolución
3. **Habla en voz alta** (nunca solo leas mentalmente)
4. **Usa cronómetro** para simular presión real
5. **Revisa tus grabaciones** y anota mejoras
6. **Usa el ejercicio interactivo HTML** para práctica cronometrada

### ❌ Evita SIEMPRE
1. No escribas guiones completos (solo palabras clave)
2. No traduzcas mentalmente (piensa en francés desde el inicio)
3. No practiques solo leyendo (debes HABLAR)
4. No te compares con otros (cada quien avanza a su ritmo)
