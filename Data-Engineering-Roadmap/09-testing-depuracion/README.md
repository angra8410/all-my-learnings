# Módulo 09 — Testing y Depuración

Objetivos:
- Implementar testing para pipelines y tests de calidad de datos.

Contenido:
1. Tipos de tests: unit, integration, e2e para pipelines
2. Data quality tests: null checks, uniqueness, ranges
3. Herramientas: pytest, Great Expectations (o dbt tests)
4. Estrategias de logging y tracing para debugging
5. Simulación de fallos y recovery

Actividades prácticas:
- Escribir tests de unidad para funciones Python y pruebas de integración.
