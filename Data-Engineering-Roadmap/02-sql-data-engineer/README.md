# Módulo 02 — SQL para Data Engineers

Objetivos:
- Dominar consultas SQL para análisis y transformación de datos.
- Aprender técnicas de optimización y window functions.

Contenido:
1. SELECT, WHERE, JOIN (INNER, LEFT, RIGHT, FULL)
2. Agregaciones y GROUP BY
3. Subconsultas y CTEs (WITH)
4. Window functions (ROW_NUMBER, RANK, PARTITION BY)
5. Índices y buenas prácticas de performance
6. SQL para pipelines: INSERT ... SELECT, MERGE / UPSERT

Actividades prácticas:
- Conjunto de ejercicios con un dataset de ventas: consultas, joins, agregaciones.
- Reto: escribir una consulta con window functions para clientes top-N.
- Actividad: escribir scripts SQL para cargar y deduplicar datos.

Duración estimada: 8-12 horas
Entregables: scripts .sql y respuestas en actividad-interactiva.md
