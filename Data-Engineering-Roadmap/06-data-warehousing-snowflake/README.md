# Módulo 06 — Data Warehousing (Snowflake)

Objetivos:
- Comprender conceptos de un data warehouse y aprender Snowflake básico.

Contenido:
1. Conceptos de DW vs data lake
2. Arquitectura de Snowflake: databases, schemas, warehouses, stages
3. Cargar datos (COPY INTO), formatos (CSV, Parquet)
4. Particionamiento, clustering, scaling
5. Autenticación, roles y permisos
6. Integración con dbt y Airflow

Actividades prácticas:
- Crear tablas en Snowflake, cargar datos y hacer queries de performance.
