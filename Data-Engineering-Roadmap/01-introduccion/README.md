# Módulo 01 — Introducción

Objetivos:
- Entender la función del Data Engineer en un equipo de datos.
- Conocer el roadmap del curso y los entregables.
- Preparar el entorno de trabajo (Git, VSCode, Docker básico).

Contenido:
1. ¿Qué hace un Data Engineer?
2. Ciclo de vida de datos: ingestión → almacenamiento → transformación → consumo
3. Herramientas clave del curso
4. Configuración del entorno (repositorio, ramas, VSCode, Docker)
5. Cómo seguir el plan de estudio y medir progreso

Actividades prácticas:
- Clonar el repositorio y crear una rama de trabajo.
- Crear un archivo progreso.md con tu plan personal (horas/semana).
- Pequeña práctica: ejecutar un contenedor Docker con una imagen de PostgreSQL local.

Duración estimada: 2-3 horas
Recursos: links a Git, Docker, plantilla de progreso.
