### OpenAI API Quick Start
```python
from openai import OpenAI

client = OpenAI(api_key="<YOUR_API_KEY>")

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello!"}
    ],
    temperature=0.7,
    max_tokens=100
)

print(response.choices[0].message.content)
```

### LangChain Quick Start
```python
from langchain_openai import ChatOpenAI
from langchain.prompts import ChatPromptTemplate

llm = ChatOpenAI(model="gpt-3.5-turbo")

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("user", "{input}")
])

chain = prompt | llm

result = chain.invoke({
    "role": "helpful assistant",
    "input": "Hello!"
})

print(result.content)
```

## üìû Getting Help

### When Stuck
1. **Read error messages carefully** - Most contain the solution
2. **Search Stack Overflow** - Likely someone had same issue
3. **Check official docs** - Often has examples
4. **Use ChatGPT/Claude** - Explain your error
5. **Ask in communities** - Be specific and show your code

### How to Ask Good Questions
- Include error messages
- Show what you've tried
- Provide minimal reproducible example
- Specify environment (OS, Python version, etc.)
