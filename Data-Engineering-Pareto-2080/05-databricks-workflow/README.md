# Módulo 05 — Databricks Workflow (Pareto 20/80)

Objetivo general:
Aprender a usar Databricks notebooks, jobs y clusters para orquestar trabajos Spark y flujos ETL, enfocando en lo esencial para producción.

Resultados de aprendizaje:
- Crear notebooks reproducibles y parametrizados.
- Configurar jobs y clusters mínimos para pruebas y dev.
- Exportar notebooks y usar Databricks CLI para automatización.

Duración estimada: 8-12 horas

Pareto 20/80:
- 20%: notebooks parametrizados, jobs, mounting de storage
- 80%: ejecutar nuts-and-bolts: notebooks que ejecuten los jobs y manejo de secrets
