# Módulo 02 — SQL Core (Pareto 20/80)

Objetivo general:
Dominar el 20% de conceptos SQL que producen el 80% del impacto: consultas complejas, window functions, joins eficientes y cargas incrementales (MERGE/UPSERT).

Resultados de aprendizaje:
- Escribir consultas eficientes con JOINs y agregaciones.
- Usar funciones de ventana para rankings y series temporales.
- Implementar MERGE/UPSERT para cargas incrementales.
- Optimizar consultas con índices y particionamiento.

Estructura del módulo:
- Teoría breve y ejemplos (20% para aprender)
- Actividades prácticas y proyectos (80% para practicar)
- Proyecto mini: conjunto de consultas para dashboard de ventas

Duración estimada: 8-12 horas
Prerequisitos: SQL básico (SELECT, WHERE), acceso a una BD local (Postgres o similar) o entorno en Docker.

Pareto 20/80 (resumen):
- 20% para aprender: SELECT complejo, JOINs, GROUP BY, window functions, MERGE.
- 80% para practicar: escribir consultas para casos reales (top-N, running totals, deduplicación), optimización y scripts de carga.

Archivos clave en este módulo:
- actividad-interactiva.md (ejercicios paso a paso)
- progreso.md (checklist)
- retroalimentacion.md (rúbrica)
- recursos.md (datasets, enlaces y comandos)
