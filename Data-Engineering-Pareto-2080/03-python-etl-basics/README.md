# Módulo 03 — Python ETL Basics (Pareto 20/80)

Objetivo general:
Aprender el 20% de Python/ETL que permite construir 80% de pipelines: lectura/escritura, transformaciones con pandas, conexión a bases de datos y manejo de errores/logging.

Resultados de aprendizaje:
- Scripts reproducibles de ingestión desde CSV y APIs.
- Transformaciones idempotentes y carga a BD.
- Tests unitarios simples para funciones de transformación.

Duración estimada: 10-14 horas

Pareto 20/80:
- 20% a aprender: manejo de pandas, logging, SQLAlchemy para conexiones, requests para APIs.
- 80% a practicar: escribir pipelines de ingestión y tests automatizados.

Estructura del módulo: actividad-interactiva.md, scripts de ejemplo en /examples, progreso, retroalimentacion, recursos.
