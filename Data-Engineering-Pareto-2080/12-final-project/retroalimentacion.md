# Retroalimentación — Módulo 12 (Proyecto Integrador)

Criterios de evaluación (detalle):
- Ejecución reproducible del pipeline (40%)  
  - ¿Se puede ejecutar localmente o en cloud con las instrucciones?  
  - ¿Los datos fluyen de raw → marts correctamente?  
- Calidad del modelado y tests (30%)  
  - Correctitud semántica, SCDs si aplican, tests en dbt/pytest/GE.  
- Documentación y entregables (20%)  
  - README-deploy claro, demo funcional, instrucciones para reviewer.  
- Observabilidad y gestión de errores (10%)  
  - Métricas mínimas, alertas y runbook.

Comentarios de revisión:
- Pide logs y captura de ejecución de un run completo.
- Solicita un video corto (3–5 min) mostrando el pipeline si es posible.
